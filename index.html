IMVIA Creative

Privacy Policy

Last updated: May 31, 2026

IMVIA Creative is a browser extension that connects supported Web creative tools with Adobe Photoshop through a local IMVIA Creative Bridge application.

Data We Process

IMVIA Creative may process the following data only when the user actively uses the extension:

  • User-selected images or creative content from supported websites.
  • Current tab information needed to identify supported websites.
  • Extension settings, license status, and local Bridge connection status.
  • Hashed device identifier generated by the local Bridge for license and trial eligibility checks.

Local Processing

User images, PSD data, SVG data, layer data, and creative files are transferred locally between the browser, the local IMVIA Creative Bridge, and Photoshop. These creative files are not uploaded to IMVIA servers.

Server Communication

IMVIA servers are used only for license activation, license verification, 7-day trial eligibility, device binding, and basic abuse-prevention checks. The server does not receive or store user creative files.

Device Identifier And Trial Eligibility

To support license verification, device binding, one 7-day trial per device, and basic abuse prevention, IMVIA uses a device identifier generated by the local Bridge. Before it is sent to the License API, the identifier is hashed.

IMVIA does not upload hardware serial numbers, MAC addresses, full machine details, or raw device identifiers. The hashed device identifier is used only for licensing, trial eligibility, and abuse prevention. It is not used for advertising tracking and is not shown in the UI or logs.

Data We Do Not Collect

IMVIA Creative does not collect:

  • Passwords.
  • Cookies or login tokens.
  • Payment information.
  • Browsing history.
  • Health information.
  • Personal communications.
  • Precise location.
  • User creative files on IMVIA servers.
  • Hardware serial numbers, MAC addresses, or raw device identifiers.

Third-Party Websites

IMVIA Creative does not bypass third-party website login, payment, upload, or usage limits. The extension only works when the user actively chooses to transfer content.

Data Sharing

We do not sell user data. We do not transfer user data to third parties except as necessary to provide the extension's approved functionality.
